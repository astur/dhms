# dhms

Parses `dhms` string to number of milliseconds

[![Build Status][travis-image]][travis-url]
[![NPM version][npm-image]][npm-url]

## Install

```bash
npm i dhms
```

## Usage

```js
const dhms = require('dhms');
dhms('1d2h30m45s123'); // 95445123
```
